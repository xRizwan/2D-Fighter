# 2D-Fighter

A 2d Platformer game, with 2 levels, one basic and one boss level, also has a local multiplayer mode. No story added yet.

i didn't do any of the art.

Controls :-

  Player 1 - Arrow keys and Z

  Player 2 - WAD and G
